# Jot - OpenSource TUI notes application

jot down (verb):
1. To write down hurriedly; to make a note of.
2. Write briefly or hurriedly; write a short note of.

Jot is a free notes app that lives and breaths the terminal. It allows for quick access and editing of different todo lists.

All of the created 'jots' are stored as pure text files that can be moved, deleted and copied however you like. No lock-in, no proprietary algorithms.

# Commands
exit -> Exit Jot
